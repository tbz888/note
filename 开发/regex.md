> Regular Expression : 按正确的规则来处理字符串
### Rule
| 符号           | 描述                                |
| ------------- | ----------------------------------------------- |
| .             | 表示任意一个字符                                |
| \             | 表示转义                                        |
| +             | 表示贪婪，前一个字符重复一次或多次              |
| *             | 表示前一个字符匹配到0次或多次 [http, https]     |
| [ ]           | 表示匹配括号内的字符中的任意一个 [字母大小写]   |
| [^ ]          | 表示排除里面的字符，其他字符都匹配              |
| ?             | 表示懒惰匹配前面，遇到结束符就停 [@sysu.edu.cn] |
| {x,y}         | 表示前一个字符重复次数[x,y]                     |
| ()            | 表示子表达式                                    |
| (?<=A)XY(?=B) | 表示匹配AXXB，只返回XY                          |
